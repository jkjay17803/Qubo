# 데이터 통신과 네트워크
qs ( )은 멀리 떨어져 있는 송신자 컴퓨터와 수신자 컴퓨터 또는 입출력장치를 통신 회선으로 연결해 데이터를 처리하고 전송하는 것을 뜻한다.
데이터 통신
qs ( )은 데이터를 전송하고 처리하는 기술을 뜻한다.
데이터 통신 기술
qs 아르파넷은 웹이라고 부르는 ( )의 시초이다.
WWW

네트워크
qs ( )이란 컴퓨터 네트워크에서 데이터를 주고받을 때 수행되는 절차이다.
프로토콜

OSI 7계층 모델
qs ( )은 OSI의 1계층에 위치하며, 호스트를 전송 매체와 연결하기 위한 인터페이스 규칙과 전송 매체의 특성을 다룬다.
물리 계층
qs ( )은 OSI의 2계층에 위치하며, 컴퓨터 단말기 등의 시스템 또는 교환기 같은 중계시스템 간 데이터 전송을 원활하게 해, 오류 검출과 회복 기능을 제공한다.
데이터 링크 계층
qs ( )은 OSI의 3계층에 위치하며, 사용자 프로그램, 단말 조작원 등 응용 프로세스 간에 데이터를 전송할 때 정보의 경로 선택과 중계 기능을 수행한다.
네트워크 계층
qs ( )은 OSI의 4계층에 위치하며, 정보를 수신자에게 보내는 운반, 운송 기능을 관리한다.
전송 계층
qs ( )은 OSI의 5계층에 위치하며, 4계층에서 제공하는 기능과 유사하지만 더 상위의 논리적 연결을 제공한다.
세션 계층
qs ( )은 OSI의 6계층에 위치하며, 정보를 데이터 표현 형식에 따라 전송한다.
표현 계층
qs ( )은 OSI의 최상위 계층에 위치하며, 정보를 처리하는 응용 프로그램 또는 프로세스의 인터페이스와 통신을 가능하게 하는 기본적인 응용 기능을 제공한다.
응용 계층
qs ( )은 일대일 통신 환경에서 데이터를 한쪽 방향으로만 전송하는 형태이다.
